｛第一：取用神：｝
｛  九星所主：天英星代表火神，主晴；天辅星代表风；天柱星代表雨；天蓬星为水神，也代表雨；天冲星代表雷；在冬季时，以天心星、天柱星代表霜雪。｝ 
｛  八门所主：以景门主晴；开、伤二门主雷；休门主云雨；生门主风；杜门主电；惊门主虹霓。 ｝
｛  天干所主：甲乙主风；丙丁主火；戊己主阴云；庚辛主冰雪；壬癸主雨。其中壬为大雨，癸为小雨。｝
｛  八神所主：以值符代表日和月；九天主晴；九地主雨；六合主风和日丽；白虎主风；玄武主雨；太阴主阴去霜雪；蛇主闪电虹霓，等等。｝
｛  按用事局时辰起局；六爻则问事局起卦、用事之日子时起卦、或摇卦皆可。｝
｛第二：详断：｝
如果 八门伏吟 或者 九星伏吟 那么  ｛  八门伏吟或九星伏呤局，主天气正常。｝
｛  1. 晴断：｝
如果 九星旺相(天英星) 并且 （天英星落宫 是 震宫  或者 天英星落宫 是 巽宫） 那么 ｛  天英星落（天英星落宫）宫旺相，落于震三、巽四宫，或克日、时二干，主晴天。｝
如果 九星旺相(天辅星) 并且 （天辅星落宫 是 离宫 或者 星克宫(天辅星落宫) 或者 二宫相克（天辅星落宫，日干落宫） 或者 二宫相克（天辅星落宫，时干落宫） ） 那么 ｛  天辅星落（天辅星落宫）宫旺相，落于离九宫，或克所落之宫，或克日、时二干，主风晴。｝
如果 九星旺相（天英星） 并且  天英星落宫 是 坎宫 并且 （天英星落宫 是 休门落宫  或者 天英星落宫  是 玄武落宫） 那么  ｛  天英星当令落坎宫，临休门或玄武，阴天。｝
如果 九星旺相（天英星） 并且  天英星落宫 是 兑宫  那么 ｛  天英星当令落兑宫，填实之日火旺金死，晴。｝
如果 天辅星落宫 是 坤宫  那么 ｛  天辅落坤2宫为入墓，不刮风。｝
如果 不是 天干旺相（壬）  并且  不是 天干旺相（癸） 或者  (天盘干入墓（壬落宫） 或者 宫克干（壬落宫） 或者 天盘干入墓（癸落宫） 或者 宫克干（癸落宫）) 那么 ｛  壬落（壬落宫）宫、癸落（癸落宫）宫，休囚或入墓或受制，皆主无雨，阴支密布之象。｝
如果 天辅星落宫 是 开门落宫  那么 ｛  天辅星落（天辅星落宫）宫逢开门，主有风。｝
如果 本宫吉格（天辅星落宫） 或者 不是 九星旺相（天辅星）  那么 ｛  天辅落（天辅星落宫）宫吉格，或者天辅星休囚，表示风小。｝
｛  2. 雨断：｝
如果 （天蓬星落宫 是 壬落宫 或者 天蓬星落宫 是 癸落宫） 并且 （天蓬星落宫 是 坎宫 或者 天蓬星落宫 是 兑宫 或者 天蓬星落宫 是 震宫） 那么 ｛    天蓬星落（天蓬星落宫）宫，乘壬、癸二干，落于坎一宫、震三宫、兑七宫，壬干主大雨，癸干主小雨。｝
如果 （天柱星落宫 是 壬落宫 或者 天柱星落宫 是 癸落宫） 并且 （天柱星落宫 是 坎宫 或者 天柱星落宫 是 兑宫 或者 天柱星落宫 是 震宫） 那么 ｛    天柱星落（天柱星落宫）宫，乘壬、癸二干，落于坎一宫、震三宫、兑七宫，壬干主大雨，癸干主小雨。｝
如果 本宫天克地（天英星落宫） 并且 本宫天克地（天辅星落宫） 那么 ｛    天英星落（天英星落宫）宫，天辅星落（天辅星落宫）宫，二宫同时天盘之干克地盘之干，必然风雨雷电交加。如与值符近，则雨速，远则雨迟。｝
如果 （天蓬星落宫 是 壬落宫 或者 天蓬星落宫 是 癸落宫） 并且 （天蓬星落宫 是 坤宫） 那么 ｛    天蓬星落坤二宫，主密云不雨。｝
如果 （天柱星落宫 是 壬落宫 或者 天柱星落宫 是 癸落宫） 并且 （天柱星落宫 是 坤宫） 那么 ｛    天蓬星落坤二宫，主密云不雨。｝
如果 （天蓬星落宫 不等于 壬落宫 并且 天蓬星落宫 不等于 癸落宫） 并且 （天蓬星落宫 是 坎宫 或者 天蓬星落宫 是 兑宫 或者 天蓬星落宫 是 震宫） 那么 ｛    天蓬星落（天蓬星落宫）宫，不乘壬癸水神，即使落于坎一宫、震三宫、兑七宫，也不会下雨，或主当时已下过雨了，或雨停了。｝
如果 （天柱星落宫 不等于 壬落宫 并且 天柱星落宫 不等于 癸落宫） 并且 （天柱星落宫 是 坎宫 或者 天柱星落宫 是 兑宫 或者 天柱星落宫 是 震宫） 那么 ｛    天柱星落（天柱星落宫）宫，不乘壬癸水神，即使落于坎一宫、震三宫、兑七宫，也不会下雨，或主当时已下过雨了，或雨停了。｝
如果 不是 本宫旺相（天柱星落宫） 并且 （天柱星落宫 是 坎宫） 并且 （天柱星落宫 不等于 壬落宫 并且 天柱星落宫 不等于 癸落宫） 那么 ｛    天柱星月令休囚，虽落坎宫，但无壬癸，最多天上有些云彩；｝
如果 （天蓬星落宫 是 壬落宫 或者 天蓬星落宫 是 癸落宫） 并且 （天蓬星落宫 是 乾宫） 那么 ｛    天蓬星落乾宫为比和，带壬癸，虽不如坎宫雨大，也主雨，其中壬为中雨，癸为小雨；｝
如果 （天柱星落宫 是 壬落宫 或者 天柱星落宫 是 癸落宫） 并且 （天柱星落宫 是 乾宫） 那么 ｛    天柱星落乾宫为比和，带壬癸，虽不如坎宫雨大，也主雨，其中壬为中雨，癸为小雨；｝
如果 （天蓬星落宫 是 壬落宫 或者 天蓬星落宫 是 癸落宫） 并且 （天蓬星落宫 是 离宫） 那么 ｛    天蓬星带壬癸，落离宫，不会下雨，阴天之象；｝
如果 （天英星落宫 是 壬落宫 或者 天英星落宫 是 癸落宫） 并且 （天英星落宫 是 坎宫） 那么 ｛    天英星带壬癸，落坎宫，不会下雨，阴天之象；｝
如果 壬落宫 是 艮宫 并且 二宫相生（开门落宫，壬落宫） 那么 ｛    壬落艮宫，但得开门之生，有雨之象；｝
如果 日支 是 辰 并且 时支 是 辰  那么 ｛    辰日辰时为水库，但辰辰相刑库必破，有雨；｝
{  3. 雪断：}
{    冬季，天心星、天柱星乘壬、癸到干、兑二宫，主雪。 }
{  4. 涨水：}
{    天蓬星、休门乘旺相之气而得三奇者，水虽涨，但不至于泛滥。}
{    天蓬星、休门乘旺相之气而值庚格，主河水壅塞泛滥高涨；如果乘甲辰壬并带有蛇，则洪水泛滥，肆虐为害。}
{    天蓬星、休门落于坤二宫、中五宫、艮八宫，水被土掩，洪水消退，或无洪水。}
{    天蓬星、休门不乘庚格与甲辰壬者，水虽旺发，不会决堤出岸为害。}
{    天蓬或休门入巽宫，巽为水道，必暴涨，外盘为肆虐时间长；}
{    巽宫天盘为客为洪水，地盘为主为长江，地生天则更涨；其对宫之地盘天干与宫支退潮期；}
{    天蓬和休门、玄武同落2、5、8外任一宫，旺相，必暴涨；}
